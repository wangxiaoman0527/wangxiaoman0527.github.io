一、JDK、JRE和JVM的区别

JVM ：负责执行Java字节码，实现Java程序的跨平台特性，是JRE的一部分。
JRE ：Java程序运行的最小环境，包含JVM和核心类库。比如你电脑上只想跑个Java写的桌面工具，装JRE就够了。
JDK ：JRE + 开发工具（比如javac、javap、jconsole这些）。你要写代码、编译、调试，就必须装JDK。
